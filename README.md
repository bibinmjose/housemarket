# House Market - Property Investment Financial Calculator

## Overview

This repository contains Excel spreadsheets for comprehensive **property investment financial analysis and planning**. The spreadsheets model real estate investment scenarios including mortgage calculations, rental income, tax benefits, capital appreciation, and long-term financial projections.

## Repository Contents

### Files

1. **`calculation.xlsx`** - Multi-property investment scenario calculator with comparative analysis
2. **`calculation_v2.xlsx`** - Enhanced version with additional financial metrics
3. **`single-calculation.xlsx`** - Detailed single property investment calculator

## Features

### 1. Mortgage & Loan Calculations
- **Amortization schedules** for 30-year loans
- Principal and interest payment breakdowns using Excel's PMT, PPMT, and IPMT functions
- Annual interest rates (5.54% - 6%)
- Loan amounts up to $607,400
- Weekly payment calculations

### 2. Property Investment Analysis
- **Capital appreciation modeling** (6% annual growth rate)
- **Initial property values** starting from $750,000
- Year-over-year property value projections
- Net capital growth calculations (property value minus outstanding loan balance)

### 3. Rental Income Projections
- Weekly rental income tracking (e.g., $660/week)
- Rental yield calculations (80% effective yield)
- Annual rental growth modeling (6% increase)
- Multi-year rental income forecasting

### 4. Tax Calculations & Benefits
- **Parameterized tax rate** (20%)
- Tax deductions on mortgage interest payments
- Tax incentive calculations for investment properties
- Comparative cashflow analysis:
  - With tax deductions
  - Without tax deductions

### 5. Financial Projections
- Present value calculations adjusted for inflation
- Net Present Value (NPV) analysis
- Long-term investment return modeling
- Multi-scenario comparison capabilities

## Key Parameters

The spreadsheets use the following key parameters (customizable):

| Parameter | Default Value | Description |
|-----------|--------------|-------------|
| Annual Interest Rate | 5.54% - 6% | Mortgage loan interest rate |
| Loan Term | 30 years | Mortgage amortization period |
| Property Value | $750,000 | Initial property purchase price |
| Loan Amount | $607,400 | Initial mortgage principal |
| Capital Appreciation | 6% | Annual property value growth |
| Rental Income | $660/week | Weekly rental income |
| Rental Yield | 80% | Effective rental yield after expenses |
| Rental Growth | 6% | Annual rental income increase |
| Tax Rate | 20% | Income tax rate for deductions |

## Use Cases

These spreadsheets are designed for:

1. **Property Investment Analysis** - Evaluate the financial viability of real estate investments
2. **Mortgage Planning** - Understand loan repayment schedules and interest costs
3. **Tax Planning** - Calculate tax benefits from property investment
4. **Portfolio Comparison** - Compare multiple property investment scenarios
5. **Long-term Financial Forecasting** - Project returns over 30+ years
6. **Retirement Planning** - Assess property as a retirement investment strategy

## How to Use

1. Download the Excel files
2. Open in Microsoft Excel or compatible spreadsheet software (Excel 2007+)
3. Modify the key parameters in the input cells to match your scenario
4. Review the calculated projections and financial metrics
5. Compare different scenarios using the multiple spreadsheet versions
